## Form Input Edge Cases

### Address mrkdwn injection and whitespace-only input

Identified actionable edge cases from the form text input and fixed the two most impactful:

1. **mrkdwn injection** — user input containing `*`, `_`, `~`, `` ` ``, or `>` would render as Slack formatting when substituted into story text. Fixed by wrapping each special character with zero-width spaces (`\u200B`) in `setFormData()`, neutralizing formatting while preserving visual appearance.

2. **Whitespace-only input** — submitting only spaces would pass Slack's built-in required check but produce empty narrative text. Fixed by validating in `formSubmitCallback` before `ack()` — returns `response_action: "errors"` to keep the modal open with an error message.

TDD: wrote 4 tests first (2 state sanitization, 2 submission rejection), confirmed failures, then implemented.

## Edge Cases as Story Branches

### Address remaining edge cases as different story paths demonstrating Block Kit

Turned four form input edge cases into distinct story branches, each showcasing a Block Kit capability:

1. **Cancel escape hatch (mixed actions block)** — `postmortem_tonight` now has both a form button and a "Skip it and go home" choice button in the same actions block. Skipping leads to `postmortem_skipped`, a cautionary ending. Extracted `buildChoiceButton()` helper for reuse.

2. **Richer modal layout (multi-block modal)** — `buildFormInputModal` now supports optional `modalHeader`, `modalFlavorText`, and `modalHint` fields. The postmortem modal shows a header (`:memo: Post-Mortem`), flavor text about #incident-review, and a brevity hint. The mentor modal shows different flavor text about Alex.

3. **Emoji in placeholder** — Updated the postmortem placeholder to `":fire: Null pointer in user profile endpoint"` and the mentor placeholder to `":star: Great debugging today!"`. Player emoji renders in the ending narrative.

4. **Second form input path (mentor_fix)** — Converted `mentor_fix` from an ending to a form input node. Player writes an encouraging message to Alex. Leads to `mentor_fix_complete` (personalized ending with `{{alexMessage}}`) or `mentor_fix_quick` (quick thanks, less satisfying ending).

Story now has 17 nodes and 8 endings (was 14/7).
